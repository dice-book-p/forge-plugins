대상 레이어 또는 서비스의 빌드를 실행하고 결과를 보고합니다.

> **프로젝트 설정 로드**: 실행 시작 시 CLAUDE.md의 `## 빌드 명령` 섹션을 Read합니다.

## 사용법

```
/forge-flow:build-check [대상]
```

빌드 대상과 명령은 CLAUDE.md의 `## 빌드 명령` 섹션에 정의되어 있습니다.
인자 없이 실행하면 `all` 대상으로 전체 빌드를 수행합니다.

## 결과 보고 형식

```
✅ [대상] 빌드 성공 (elapsed: Ns)
❌ [대상] 빌드 실패
  - 오류: [에러 요약]
  - 파일: [관련 파일]
```

## 테스트 실행

CLAUDE.md에 `## 테스트 명령` 섹션이 정의되어 있으면, 빌드 성공 후 테스트도 실행합니다.
정의되어 있지 않으면 테스트 단계를 스킵합니다.

```
✅ [대상] 빌드 성공 (elapsed: Ns)
✅ [대상] 테스트 성공 (elapsed: Ns)
❌ [대상] 테스트 실패
  - 실패 테스트: [테스트명]
  - 오류: [에러 요약]
```

## 주의사항

- `all` 옵션은 빌드/테스트 실패 시 중단 없이 전체 결과를 보고
- 테스트 포함 빌드를 기본으로 함 (스킵 옵션 사용 지양)
