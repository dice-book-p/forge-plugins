Frontend 배포 전 품질 검증을 실행합니다. 타입체크 → 린트 → 빌드 순서로 진행합니다.

> **프로젝트 설정 로드**: 실행 시작 시 CLAUDE.md의 `## 빌드 명령` 섹션을 Read합니다.

## 사용법

```
/forge-flow:fe-check [대상]
```

FE 앱 목록과 검증 명령은 CLAUDE.md의 `## 빌드 명령` 섹션에 정의되어 있습니다.
인자 없이 실행하면 `all` 대상으로 전체 앱을 검증합니다.

## 검증 단계

타입체크 → 린트 → 빌드 순서로 실행합니다.
- 타입체크 실패 시 → 린트 스킵 → 빌드 스킵
- 린트 실패 시 → 빌드 스킵

## 결과 보고 형식

```
[FE 품질 검증 결과 - {대상}]
✅ 타입체크 통과
✅ 린트 통과
✅ 빌드 성공

배포 준비 완료
```

```
[FE 품질 검증 결과 - {대상}]
✅ 타입체크 통과
❌ 린트 실패 (N개 오류)
  - 파일:라인 - 규칙명
⏭️ 빌드 스킵 (린트 실패로 중단)

수정 후 재실행 필요
```

## 주의사항

- 각 단계 오류는 파일명/라인 수준으로 요약
